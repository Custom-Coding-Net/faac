FAAD is a codec plugin for Cooledit
to import .aac/.mp4 files and to export in AAC format.

To use it:
----------

In visual studio set "Active Configuration = FAAD - win32 Release" and compile,
copy FAAC.flt into CoolEdit folder
delete flt.dat
